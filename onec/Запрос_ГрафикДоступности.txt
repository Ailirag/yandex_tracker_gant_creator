// График доступности ресурса по группам СА и разработчиков (FTE по дням).
//
// Назначение: выгрузить реальную доступность (с учётом отпусков) для передачи
// в планировщик задач как источник ёмкости вместо мока.
//
// Основа расчёта (ВТ_Дни / ВТ_Итог / ВТ_Результат) — проверенный запрос
// «график доступного ресурса», без изменений. Изменён только финальный
// разрез: агрегируем по роль-группе и добавляем Роль и Направление.
//
// Параметры (задать в консоли запросов):
//   &НачалоПериода    — дата начала (напр. 2026-07-15), тип Дата
//   &ОкончаниеПериода — дата конца горизонта (напр. 2026-12-31), тип Дата
//
// Отличия от исходника (осознанные):
//   1. Убран фильтр «Сотрудник В ИЕРАРХИИ (&Сотрудники)» — отбор идёт по роли
//      в финале, поэтому параметр &Сотрудники больше не нужен.
//   2. Исключена группа «Разработчики второго уровня ТП» (2-я линия ТП, вне
//      планового пула). Чтобы включить — удалите соответствующую строку в ГДЕ.
//
// Выгрузка результата: колонки Дата, Группа, Роль, Направление, ДоступныйРесурс.
// Сохранить как .xlsx (или .csv) и передать в планировщик:
//   planner plan --capacity file --capacity-file <файл>

ВЫБРАТЬ
	ДанныеПроизводственногоКалендаря.Дата КАК Дата
ПОМЕСТИТЬ ВТ_Дни
ИЗ
	РегистрСведений.ДанныеПроизводственногоКалендаря КАК ДанныеПроизводственногоКалендаря
ГДЕ
	ДанныеПроизводственногоКалендаря.Дата МЕЖДУ &НачалоПериода И &ОкончаниеПериода
;

////////////////////////////////////////////////////////////////////////////////
ВЫБРАТЬ
	Дни.Дата КАК Дата,
	ДоступностьРесурсаОстаткиИОбороты.Сотрудник КАК Сотрудник,
	СУММА(ВЫБОР
			КОГДА ДоступностьРесурсаОстаткиИОбороты.Период = &НачалоПериода
				ТОГДА ДоступностьРесурсаОстаткиИОбороты.ДоступныйРесурсКонечныйОстаток
			ИНАЧЕ ВЫБОР
					КОГДА ДоступностьРесурсаОстаткиИОбороты.Период <= Дни.Дата
						ТОГДА ДоступностьРесурсаОстаткиИОбороты.ДоступныйРесурсОборот
					ИНАЧЕ 0
				КОНЕЦ
		КОНЕЦ) КАК ДоступныйРесурс
ПОМЕСТИТЬ ВТ_Итог
ИЗ
	ВТ_Дни КАК Дни,
	РегистрНакопления.ДоступностьРесурса.ОстаткиИОбороты(, , День, , ) КАК ДоступностьРесурсаОстаткиИОбороты

СГРУППИРОВАТЬ ПО
	Дни.Дата,
	ДоступностьРесурсаОстаткиИОбороты.Сотрудник
;

////////////////////////////////////////////////////////////////////////////////
ВЫБРАТЬ
	ВТ_Итог.Дата КАК Дата,
	ВТ_Итог.Сотрудник КАК Сотрудник,
	ВЫБОР
		КОГДА ВТ_Итог.ДоступныйРесурс < 0
			ТОГДА 0
		ИНАЧЕ ВТ_Итог.ДоступныйРесурс
	КОНЕЦ КАК ДоступныйРесурс
ПОМЕСТИТЬ ВТ_Результат
ИЗ
	ВТ_Итог КАК ВТ_Итог
;

////////////////////////////////////////////////////////////////////////////////
ВЫБРАТЬ
	ВТ_Результат.Дата КАК Дата,
	ВТ_Результат.Сотрудник.Родитель.Наименование КАК Группа,
	ВТ_Результат.Сотрудник.Родитель.Роль КАК Роль,
	ВТ_Результат.Сотрудник.Родитель.Родитель.Наименование КАК Направление,
	СУММА(ВТ_Результат.ДоступныйРесурс) КАК ДоступныйРесурс
ИЗ
	ВТ_Результат КАК ВТ_Результат
ГДЕ
	ВТ_Результат.Сотрудник.Родитель.Роль В (
			ЗНАЧЕНИЕ(Перечисление.ВидыСотрудников.Аналитик),
			ЗНАЧЕНИЕ(Перечисление.ВидыСотрудников.Разработчик))
	И ВТ_Результат.Сотрудник.Родитель.Наименование <> "Разработчики второго уровня ТП"

СГРУППИРОВАТЬ ПО
	ВТ_Результат.Дата,
	ВТ_Результат.Сотрудник.Родитель.Наименование,
	ВТ_Результат.Сотрудник.Родитель.Роль,
	ВТ_Результат.Сотрудник.Родитель.Родитель.Наименование

УПОРЯДОЧИТЬ ПО
	Дата,
	Группа
